HTML (Hypertext Markup Language) can be used for making a website. HTML uses tags to create this content. Here is a basic HTML document.

<!doctype html>
<html>
  <head>
    <title>Hello, world!</title>
  </head>
  <body>
    <p>Hello, world!</p>
  </body>
</html>
